Root Measure - Installation Instructions
========================================

Requires macOS (Apple Silicon or Intel) or Windows 10+.
The installer will set up Python 3.12, dependencies, and the app automatically.

Open the Terminal (Mac) or PowerShell (Windows) and paste the command below.


Mac
---
curl -sL https://raw.githubusercontent.com/williangviana/root-measure/stable/install/install.sh | bash


Windows
-------
powershell -ExecutionPolicy Bypass -c "irm https://raw.githubusercontent.com/williangviana/root-measure/stable/install/install.ps1 | iex"


After installation:
  Mac     - Open "Root Measure" from Applications or Launchpad.
  Windows - Use the "Root Measure" shortcut on your Desktop.

Note for managed Macs (no admin access):
  The installer will automatically use system Python and install to
  ~/Applications instead of /Applications. You can open the app from
  Finder → your home folder → Applications.
